# 🚀 SalesPro Technologies - AI Sales Intelligence & Lead Management System

An AI-powered Lead Management System developed using **Python, Flask, SQLite, HTML, CSS, and JavaScript** to help sales teams manage leads, compare companies, analyze sales data, and make data-driven business decisions.

---

## 📌 Project Overview

SalesPro Technologies is a web-based CRM application that allows users to:

- Manage company leads
- Store company details
- Track financial performance
- Compare multiple companies
- View AI-based sales insights
- Analyze business performance through dashboards and charts

The system helps sales teams prioritize leads based on lead scores, buying intent, financial status, and business performance.

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/srivarsha51/SalesPro_Lead_Management.git
```

### Go to Project Folder

```bash
cd SalesPro_Lead_Management
```

### Install Flask

```bash
pip install flask
```

### Create Database

```bash
python create_db.py
```

### Run Project

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
